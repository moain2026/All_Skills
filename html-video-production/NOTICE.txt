Bundled upstream skills
-----------------------
The following upstream skills, located under `skills/<name>/` in the upstream
repository at the pinned commit, are redistributed under
`references/<name>/` in this package:

  - gsap
  - hyperframes
  - hyperframes-cli
  - hyperframes-registry

The upstream skills `animejs`, `css-animations`, `lottie`, `three`, `waapi`,
`tailwind`, and `remotion-to-hyperframes` are intentionally not bundled.

Modifications
-------------
A small number of upstream files were modified to replace third-party
media-generation paths (Kokoro TTS, `npx hyperframes tts`,
`npx hyperframes transcribe`, ElevenLabs, HeyGen TTS, Whisper) with
Manus-native tools (`generate_speech`, `generate_music`, `generate_image`)
and a sandbox utility (`manus-speech-to-text`). Each modified file carries a
prominent "Modified by Manus" notice. The complete diff summary is provided
in `_manus-overrides/modifications.md`, satisfying the modification
disclosure requirement of Apache License 2.0 §4(b).

The following Manus-authored materials are added in this package:

  - SKILL.md (top-level)
  - LICENSE
  - NOTICE.txt (this file)
  - _manus-overrides/media-generation.md
  - _manus-overrides/modifications.md
